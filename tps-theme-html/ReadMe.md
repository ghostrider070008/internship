Перед запуском должен быть установлен NodeJS, npm, Webpack и Webpack-CLI глобально.
	1. git clone https://github.com/ghostrider070008/internship.git   // Выкачиваем проект из репозитория
	2. cd ./internship/tps-theme-html	// Спускаемся в корень проекта
	3. npm i             		// Устанавливаем модули
	4. npm audit fix 		// Фиксим расхождения зависимостей       
	5. git status 			// Убедились что на ветке мастер (то что нечего коммитить, нет изменений в рабочем каталоге/конфликтов)
	6. npm run build                // Запускаемся с Билды
	7. npm start			// Разворачиваем сервер для работы над проектом.
